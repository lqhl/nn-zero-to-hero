# 前言

本书是 [nn-zero-to-hero](https://github.com/karpathy/nn-zero-to-hero) 课程的中文笔记合集，由 Qin Liu 根据课程视频和 Jupyter Notebook 整理生成。

源文件地址：https://github.com/lqhl/nn-zero-to-hero

课程内容涵盖：

1. **Lecture 1**：神经网络与反向传播入门——从零构建 micrograd
2. **Lecture 2**：逐步拆解语言建模——构建 makemore
3. **Lecture 3**：用 MLP 构建字符级语言模型（makemore Part 2）
4. **Lecture 4**：激活值、梯度与批归一化（Activations, Gradients, BatchNorm）
5. **Lecture 5**：手动反向传播——成为反向传播忍者
6. **Lecture 6**：构建 WaveNet —— 层次化卷积语言模型
7. **Lecture 7**：从零构建 GPT
8. **Lecture 8**：从零构建 GPT Tokenizer
